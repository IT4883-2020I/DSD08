# DSD_08
Phát triển phần mềm phân tán 20201 - Nhóm 08

### ENV backend

```
DB_CONNECTION=pgsql
DB_HOST=ec2-174-129-205-197.compute-1.amazonaws.com
DB_PORT=5432
```

### Install Dependencies
```
cd frontend
npm install
cd backend
composer install
```

### Run
```
cd frontend
npm start
cd backend
php artisan serve
```
